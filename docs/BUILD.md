# Build The Module:
- use the command:
  `scons platform=[your platform]`
    where [your platform] is the platform youre building for
    Ex: windows, ios, macos
- wait...
- done

# Find & Use Module:
New files for the module is in demo/bin, copy the bin folder into any project folder (where you see the project.godot) and paste. Then the project will have the `GrapplingHook` class for your use.

Find the `DOCUMENTATION.md` file in the same folder as this one for quick brief on how to use the methods in `GrapplingHook`.
